## 📖 Sobre o projeto

Este projeto consiste em um **front-end** para um sistema de rastreamento de pedidos.

O usuário informa o **ID do pedido** e é direcionado para uma página onde será exibida a localização da entrega através do **Google Maps**.

> **Status do projeto:** 🚧 Em desenvolvimento.

## ✨ Funcionalidades

* 📦 Campo para informar o ID do pedido
* ➡️ Redirecionamento para a página de rastreamento
* 🗺️ Estrutura preparada para integração com o Google Maps
* 📱 Interface responsiva

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Node

## 💻 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/joaquimgomes8/EmManutencao_site_rastreamento_pedidos/
```

Entre na pasta do projeto:

```bash
cd EmManutencao_site_rastreamento_pedidos
```

Abra o arquivo `index.html` em seu navegador.
Ou pelo terminal: `node server.js`
